Bible Online

Bible reading made easy and accessable

About the project

Bible online is a website containing the bible online, with beautiful calligraphical fonts, and full customizability. The project is also open source

Benefits of this project

Easy to use

All you got to do is just click one button, and you can start reading

Customizable

Want to change colors? No problem, we got 6 colors to chose from. Want a more historical look? No problem, we got plenty to chose from. Want a modern look? That's our initial look

Multilingual

Want to read the bible in its original? We got it. Want to read the bible in Arabic? We got it. Want to read the bible in English? No problem

Translations available:

English: New International Version (work in progres) Arabic: Kitab Al-Hayat (will come in the future)
